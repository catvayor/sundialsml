val make_sens : ?context:Sundials.Context.t ->
       int ->
       ('d, 'k) Nvector.t -> ('d, 'k, 's, [ `Sens ]) Sundials_NonlinearSolver.t

Creates a nonlinear solver based on Newton's method for sensitivity-enabled integrators. Solves nonlinear systems of the form $F(y) = 0$ .

In the call make_sens count y,

  • count is the number of vectors in the nonlinear problem, if there are $N_s$ sensitivities, then count should be $N_s + 1$ if using a simultaneous corrector or $N_s$ if using a staggered corrector; and,
  • y is a template for cloning vectors.
  • See Sundials: SUNNonlinSol_Newton
