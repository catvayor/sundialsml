Module Nvector_many.Any

module Any: sig .. end

A generic nvector interface to many-vector nvectors.

Create many-vector nvectors using the generic nvector interface where the payload is wrapped with the Many constructor.

val wrap : ?context:Sundials.Context.t -> Nvector.any Sundials.ROArray.t -> Nvector.any

Creates a generic nvector from an array of generic nvectors.

  • See Sundials: N_VNew_ManyVector
val unwrap : Nvector.any -> Nvector_many.data

Returns the payload of the generic vector if it was constructed with Many, otherwise raises Nvector.BadGenericType.
